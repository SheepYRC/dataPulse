### 1. SQL 实验室 (SQL Workbench)
- **文件导入**: 支持 CSV/Excel/Parquet 高速导入至 DuckDB。
- **查询执行**: 实时 SQL 执行，结果通过 `st.dataframe` 展示（支持大规模数据）。
- **快照保存**: 点击“保存为结果快照”可将当前结果持久化为 `.parquet` 文件。

### 2. 数据资产 (Data Assets)
- **存储审计**: 展示分析数据库的体积。
- **元数据查询**: 实时查看表结构、行数及字段定义。
- **健康检查**: 统计各字段的空值分布。

### 3. 历史与快照 (History & Snapshots)
- **持久化日志**: 使用 SQLite 记录所有执行过的 SQL。
- **文件管理**: 自动罗列并支持手动删除 `data/snapshots/` 下的 Parquet 文件。

### 4. 首页与可视化 (Home & Insights)
- **快捷动作**: 集成了缓存清理、数据库备份及快速跳转。
- **动态图表**: 在可视化探索模块中，可直接对上次查询结果进行折线、柱状、散点绘图。

## 验证步骤

### 手动验证
1. **启动程序**: `uv run streamlit run src/main.py`
2. **导入测试**: 在 SQL 实验室上传一个 CSV 文件，自定义表名。
3. **查询测试**: 输入 `SELECT * FROM [表名]` 并运行，观察结果展示。
4. **生成快照**: 运行查询后点击保存，在“历史与快照”页面查看文件。
5. **绘图测试**: 运行查询后跳转至“可视化探索”，选择字段进行绘图。
